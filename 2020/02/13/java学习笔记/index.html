java学习笔记

2020-02-13
java

词法结构

java程序是用Unicode字符集编写的，java编程语言使用UTF-16编码来表示16位代码单元系列中的文本，少数API（主要在Character类中）使用32位整数将码点表示成独立的实体。java平台提供了用于在两种表示之间进行转换的方法。除了注释，标识符，以及字符和字符串的内容之外，程序中的所有输入元素都只是用ASCII字符[以及产生ASCII字符的Unicode转义符号]构成的。ASCII是美国信息交换标准码。Unicode字符编码的前128个字符就是ASCII字符。

关键字const和goto是保留的，即使他们当前未使用也是如此。如果这些c++关键字不正确地出现在程序中，这可以允许java编译器产生更好的错误信息。虽然true和false看上去可以作为关键字，但是从技术上来讲他们是布尔值。同样虽然null看上去可以作为关键字，但是从技术上讲它是空值。

如果整数带有字母L或者l，则它是long类型，否则它就是int类型。如果一个浮点数带有字母F或者f，则它是float类型，否则它就是double类型，而且可以带上ASCII字母D或者d这样的后缀。

如果非0浮点值过大，使得在四舍五入转换成其内部表示时，它变成了一个IEEE 754无穷大的数，那么就会发生编译错误。程序通过使用诸如1f/0f或-1d/0d之类的常量转换，或者通过使用Float类和Double类预定义的常量POSITIVE_INFINITY和NEGATIVE_INFINITY，可以表示无穷大的数，而不会产生编译时错误。

如果非0浮点数过小，使得在四舍五入转换成起内部表示时，它变成0，那么就会发生编译错误。如果非0浮点值很小，但是在四舍五入转换成其内部表示时，它变成了一个非0的不正常的数字，那么将不会发生编译时错误。用于表示非数字值的预定义常量在类Float和Double中被定义为Float.NaN和Double.Nan。

每个字符串值都是对String类的一个实例的引用。String对象具有一个常量值。使用方法String.intern，使得字符串值–或者更一般地讲，作为常量表达式的值的字符串–都是被”限定的”，以共享独特的实例。

类型，值和变量

java编程语言的类型分两类：基本类型和引用类型。基本类型有布尔型和整数型。数值型有整型byte，short，int，long和char，以及浮点型float和double。引用类型有类类型，接口类型和数组类型。还有一个特殊的空类型。对象是动态创建的类类型的实例或者动态创建的数组。引用类型的值是对对象的引用。所有对象（包括数组）都支持Object类的方法。字符串值通过String对象表示。

整型有byte，short，int和long，它们的值分别是8位，16位，32位和64位带符号2的补码的整数，整型还包括char，它的值是16位无符号整数，表示UTF-16代码单元。浮点型有float，它的值包括32位IEEE 754浮点数，还有double，它的值包括64位IEEE 754浮点数。boolean类型正好有两个值：true和false。

Object的成员如下：

  • 方法getClass返回Class对象
  • 方法toString返回对象的String表示
  • 方法equals和hashCode在诸如java.util.Hashtable之类的散列表中非常有用。方法equals定义了对象相等性的概念，它基于值（而不是引用）比较。
  • 方法clone用于制作对象的副本。
  • 方法wait，notify和notifyAll用于使用线程的并发编程。
  • 方法finalize用于某个对象刚好要被销毁之前调用

变量的初始值：

  • 每个类变量，实例变量或数组元素都会在创建时用默认值对其进行初始化。
  • 类型byte，short，int，long，float，double默认0
  • 类型char，默认值是空字符’\u0000’
  • 类型boolean，默认值是false
  • 所有引用类型，默认值是null

转换和提升

装箱

装箱转换把基本类型的值转换成相应的引用类型的值。

  • boolean p到Boolean r，满足r.booleanValue() == p
  • byte p到Byte r，满足r.byteValue() == p
  • char p到Character r，满足r.charValue() == p
  • short p到Short r，满足r.shortValue() == p
  • int p到Integer r，满足r.intValue() == p
  • long p到Long r，满足r.longValue() == p
  • float p到Float r，如果p不是NaN，那么r.floatValue() == p，否则r.isNaN() == true
  • double p到Double r，满足p不是NaN，那么r.doubleValue() == p，否则r.isNaN() == true

如果被装箱的p值是true，false，一个byte，一个在\u0000~\u007f之间的char，或者一个在-128~127之间的int或short数字，设r1和r2是p的任何两个装箱转换的结果，则始终有r1 == r2。如果某一包装器（Boolean，Byte，Character，Short，Integer，Long，Float或Double）的新实例需要被分配并且可用内存不足，那么装箱转换可能导致一个OutOfMemoryError。

拆箱

拆箱转换把引用类型的值转换成相应的基本类型的值。

  • Boolean r转换成r.booleanValue()
  • Byte r转换成r.byteValue()
  • Character r转换成r.charValue()
  • Short r转换成r.shortValue()
  • Integer r转换成r.intValue()
  • Long r转换成r.longValue()
  • Float r转换成r.floatValue()
  • Double r转换成r.doubleValue()
  • 如果r是null，那么拆箱转换将抛出一个NullPointerException

名称

包名称

包名称的第一个标识符包含两个或三个小写字母，它们指定了一个Internet域，如com，edu，gov，mil，net，org；或者是两个字母的ISO国家代码，如uk或jp。

仅打算提供本地使用的包的名称应该具有以小写字母开头的第一个标识符，但是第一个标识符明确地不应该是标识符java，开始于标识符java的包名称是Sun保留的，用于命名Java平台包。

类和接口类型名称

类和接口类型名称应该是说明性的名词或名词短语，不要使名称过长，采用大小写混合的形式，并且大写每个单词的首字母。

容器类型应该为其元素类型使用名称E。映射应该为其键的类型使用K，为其值的类型使用V。名称X应该用于任意的异常类型。当无需关于某个类型的更多知识即可区分它们时，我们应该为类型使用T。如果有多个指示任意类型的类型参数，则其中一个应该使用字母表中T的邻近字母，如S。一种替代方法是，使用数字下标（例如，T1，T2）来区分不同的类型变量。

方法名称

方法名称应该是动词或动词短语，采用大小写混合的形式，名称的首字母小写，而任何后续单词的首字母则大写。

  • 应该把get和set一个可能被试作变量V的熟悉的方法命名为getV和setV。例如类Thread的方法getPriority和setPriority。
  • 应该把返回某个对象长度的方法命名为length
  • 对于测试关于某个对象的boolean条件V的方法，应该命名为isV。例如类Thread的方法isInterrupted。
  • 对于将其对象转换成一种特殊格式F的方法，应该命名toF。例如类Object的方法toString，类java.util.Date的方法toLocaleString和toGMTString。

常量名称

接口类型中的常量名称应该是一个或多个单词，缩写词或简写词的序列，它们全部大写，各个成分之间用下划线”_”字符隔开，类类型的final变量传统上也可以采用这样的命名方式。常量名称应该具有说明性，不应该不必要的进行简写。例如类Character的MIN_VALUE, MAX_VALUE, MIN_RADIX和MAX_RADIX

protected字段访问控制

类Point3d中的delta方法会发生编译错误：它不能访问其参数p的protected成员x和y，这是由于虽然Point3d（在该类中发生了对字段x和y的引用）是Point（在该类中声明了x和y）的子类，但是它未包括在Point（参数p的类型）的实现中。方法delta3d可以访问其参数q的protected成员，这是由于类Point3d是Point的子类，并且包括在Point3d的实现中。
